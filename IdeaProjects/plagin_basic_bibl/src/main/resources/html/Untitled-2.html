Базовые библиотеки Java

Библиотека — это готовый набор функций и объектов для какого-либо языка программирования. Библиотека в Java — это набор готового кода (классов, методов и интерфейсов), который упрощает разработку, предоставляя функциональность для решения определенных задач (работа с файлами, сетью, базами данных и т. д.). Библиотеки подключаются как зависимости, обычно в виде JAR-файлов, либо через системы управления зависимостями, такие как Maven или Gradle.

Java Standard Library (JDK)

Встроенная библиотека Java, содержит базовые API (коллекции, работа с файлами, строки, и т. д.) Основное назначение: базовые операции для всех приложений.
